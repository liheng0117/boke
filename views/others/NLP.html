    # （1）、Python3

    • brew 安装python3 环境
     brew install python3
    1

    一行命令搞定，但是它会安装最新版本对python3,现在mac 安装的是python3.7,有些功能模块，对最新的版本并不支持。

    • 官网下在python3 指定版本安装

    下载地址：https://www.python.org/downloads/release/python-367/ (opens new window)

    按照流程安装即可，会默认配置python3 环境。
